Netflix Clone

A responsive Netflix landing page clone built using HTML5 and CSS3. This project recreates the look and feel of Netflix's homepage while focusing on modern UI design, responsive layouts, and frontend development best practices.

Features

- Responsive Design for Desktop and Mobile Devices
- Modern Hero Section
- Netflix-inspired User Interface
- Email Signup Form
- Navigation Bar with Language Selector
- Feature Sections
- FAQ Section
- Clean and Organized Code Structure

Technologies Used

- HTML5
- CSS3
- Flexbox
- Responsive Design
- Media Queries
